# Планировщик мероприятий

Веб-приложение для планирования и управления личными мероприятиями.

---

## 🚀 Демо (работающее приложение)

👉 **Открыть приложение** 👈

> *Примечание: приложение запущено на локальном сервере. Для просмотра скопируйте репозиторий и выполните инструкции по запуску.*

---

## 📋 Основные возможности

| Функция | Описание |
|---|---|
| ➕ **Добавление** | Добавление мероприятия с указанием названия, даты/времени и описания |
| ✏️ **Редактирование** | Изменение любого мероприятия |
| 🗑️ **Удаление** | Удаление мероприятия из списка |
| ✓ **Выполнено** | Отметка мероприятия как выполненного (зеленый фон) |
| 📅 **Сортировка** | Мероприятия сортируются по дате (ближайшие сверху) |
